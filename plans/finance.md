# Financial & Payment Module

## Automated Reconciliation
- **Transaction Mapping:** When a parent pays via digital wallets (e.g., eSewa/Khalti), the remark containing the Student ID is matched automatically.
- **Ledger:** Real-time credit/debit tracking for every student.

## Accountant Workflow
- **Verification:** Accountants only review flagged or "manual upload" receipts.
- **Billing:** Automated monthly tuition generation based on class-specific fee structures.
