# UI/UX & Motion Design

## Visual Language
- **Colors:** Deep Blues (#1D4ED8) for Trust, Emeralds for Finance, Gold for Achievements.
- **Typography:** Inter/Sans-serif with large tap targets (minimum 44x44px).

## Animations
- **Splash:** Smooth logo fade-in with a scale-up effect.
- **Transitions:** Horizontal slides between views to mimic native Android Activity transitions.
- **OTP:** Pulsing focus ring on the active digit box.
