# Udayashree School Management System - Core Roadmap

## 1. Multi-Role Architecture
- [ ] **Unified Login:** Single entry point for all roles (Parent, Teacher, Admin, Accountant, Principal).
- [ ] **Role-Based Access Control (RBAC):** Restrict views based on login type.
- [ ] **State Management:** Use a more robust user object to handle permissions.

## 2. Admin Module (The Control Center)
- [ ] **User Management:** Add/Edit Student and Teacher profiles.
- [ ] **Academic Setup:** Create Classes, Sections, and assign Subject Teachers to Time Slots.
- [ ] **ID Generation:** Assign unique IDs and handle demographic data (Blood Group, Address).

## 3. Teacher Module (The Classroom Assistant)
- [ ] **Digital Register:** Live attendance marking for assigned classes.
- [ ] **Exam Portal:** Enter marks for Mid-terms and Finals.
- [ ] **Class Swap:** Request system to switch periods with other teachers.

## 4. Parent Module (The Window to the Ward)
- [ ] **Academic Progress:** View Exam Results and Digital Certificates.
- [ ] **Financials:** View generated bills and payment receipts.
- [ ] **Real-time Updates:** Homework (simplified) and Notice notifications.

## 5. Accountant & Principal Modules
- [ ] **Accountant:** Bill generation and payment verification.
- [ ] **Principal:** School-wide analytics (Total attendance, Fee collection status, Teacher performance).

## 6. Communication & Future
- [ ] **Internal Chat:** Dedicated teacher-to-teacher and teacher-to-admin messaging.
- [ ] **Firebase Sync:** Migrate all local mock data to real-time cloud storage.
